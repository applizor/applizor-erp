Applizor Softech LLP – Complete Custom ERP / CRM / HRMS / Accounting Software Blueprint (FINAL)
This document is the FINAL, COMPLETE, and SINGLE SOURCE OF TRUTH for the custom software being built for Applizor Softech LLP. It includes ALL business requirements, ALL modules, ALL technical constraints, and a PHASE-WISE development plan. This document is Cursor-ready and developer-ready.
1. Project Vision & Objective
•	Build a single unified custom software to run the complete IT services business.
•	Replace multiple tools (CRM, HRMS, Accounting, Docs) with one system.
•	Provide enterprise-grade document generation with selectable letterhead behavior.
•	Ensure scalability, audit safety, and long-term maintainability.
